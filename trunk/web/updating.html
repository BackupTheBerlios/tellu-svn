Update from 3.0.0 to 3.0.1

Estimated service maintenance break for upgrading 3.0.0 to 3.0.1 is about 15 minutes.

Purpose of this upgrade

Tellu Skin 3.0.1 requires server version 3.0.1, so you need to upgrade both, the server package and the Skin.

Upgrading the servers

First, stop running servers and back up your existing server configuration files, you need them later. Use method like this for example:

 $ /etc/init.d/tellu.init.d stop
 $ cd PREFIX/etc/
 $ mv tellud.conf tellud.conf.300
 $ mv telmond.conf telmond.conf.300
 $ mv telskind.conf telskind.conf.300

Now compile and install new server package as you did when you installed Tellu servers for first time. See compiling instructions here. All you have to do is to compile and install new servers in place, do not reinitialize the database. After succesful installation, review new configuration files in `PREFIX/etc' and change them to suit your environment, use old configuration files as a reference. Then, start new servers and proceed to upgrading the Skin.

Upgrading the Skin

Install new Skin as described here. It is not absolutely necessary to back up old Skin installation first (although it is highly recommended), you can install new Skin overwriting the old one.
