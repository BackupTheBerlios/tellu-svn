Frequently Asked Questions

Where are all the docs?

As Tellu is one man's project, work is focused mainly on coding new features and bugfixing. I will write some user guides when I have time for that, but unfortunately for now there is no any documentation except what you can find in these pages.

What does word Tellu mean?

Tellu is a finnish girl's nickname, so it is not a contortion from english word Tell'u or Tell you. Also in Peanuts comic, Lucy van Pelt is translated in finnish as Tellu.
